  • 81. Write a Java program to count total number of lines from a string 
    import java.util.Scanner;
            
            public class CountLines {
                public static void main(String[] args) {
                    Scanner scanner = new Scanner(System.in);
                    System.out.println("Enter a string (use '.' to end):");
                    StringBuilder sb = new StringBuilder();
                    String line;
                    while (!(line = scanner.nextLine()).equals(".")) {
                        sb.append(line).append("\n");
                    }
                    
                    String text = sb.toString();
                    int lines = text.split("\n").length;
                    System.out.println("Total number of lines: " + lines);
                }
            }

    Output: Enter a string (use '.' to end): Hello World, Java is great., ., Total number of lines: 2

  • 87. Write a Java program to insert an element (specific position) into an array 
    import java.util.Arrays;
            import java.util.Scanner;
            
            public class InsertElement {
                public static void main(String[] args) {
                    int[] array = {1, 2, 3, 4, 5};
                    Scanner scanner = new Scanner(System.in);
                    System.out.print("Enter the element to insert: ");
                    int element = scanner.nextInt();
                    System.out.print("Enter the position to insert (0-5): ");
                    int position = scanner.nextInt();
                    
                    if (position < 0 || position > array.length) {
                        System.out.println("Invalid position!");
                        return;
                    }
            
                    int[] newArray = new int[array.length + 1];
                    for (int i = 0; i < position; i++) {
                        newArray[i] = array[i];
                    }
                    newArray[position] = element;
                    for (int i = position; i < array.length; i++) {
                        newArray[i + 1] = array[i];
                    }
            
                    System.out.println("Array after insertion: " + Arrays.toString(newArray));
                }
            }

    Output: Enter the element to insert: 10, Enter the position to insert (0-5): 2, Array after insertion: [1, 2, 10, 3, 4, 5]

  • 88. Write a Java program to reverse an array of integer values 
    import java.util.Arrays;
            
            public class ReverseArray {
                public static void main(String[] args) {
                    int[] array = {1, 2, 3, 4, 5};
                    int n = array.length;
            
                    for (int i = 0; i < n / 2; i++) {
                        int temp = array[i];
                        array[i] = array[n - i - 1];
                        array[n - i - 1] = temp;
                    }
            
                    System.out.println("Reversed array: " + Arrays.toString(array));
                }
            }

    Output: Reversed array: [5, 4, 3, 2, 1]

                public static void main(String[] args) {

                    for (int[] row : result) {
                        for (int val : row) {
                            System.out.print(val + " ");
                        }
                        System.out.println();
                    }
                }
            }

    Output: Resultant Matrix:
    8 10 12
    5 7 9

  • 97. Write a Java program to segregate all 0s on left side and all 1s on right side of a given array of 0s and 1s 
    public class SegregateZerosOnes {
                public static void main(String[] args) {
                    int[] array = {0, 1, 0, 1, 0, 1};
                    int left = 0, right = array.length - 1;
            
                    while (left < right) {
                        while (array[left] == 0 && left < right) {
                            left++;
                        }
                        while (array[right] == 1 && left < right) {
                            right--;
                        }
                        if (left < right) {
                            int temp = array[left];
                            array[left] = array[right];
                            array[right] = temp;
                        }
                    }
            
                    System.out.println("Array after segregation: " + Arrays.toString(array));
                }
            }

    Output: Array after segregation: [0, 0, 0, 1, 1, 1]

  • 98. Write a Java program to cyclically rotate a given array clockwise by one 
    public class RotateArray {
                public static void main(String[] args) {
                    int[] array = {1, 2, 3, 4, 5};
                    int last = array[array.length - 1];
            
                    for (int i = array.length - 1; i > 0; i--) {
                        array[i] = array[i - 1];
                    }
                    array[0] = last;
            
                    System.out.println("Array after rotation: " + Arrays.toString(array));
                }
            }

    Output: Array after rotation: [5, 1, 2, 3, 4]
